# promo-tariff-cards

## Dependencies

* Jquery
* [User JavaScript and CSS Chrome Extension](https://chromewebstore.google.com/detail/user-javascript-and-css/nbhcbdghjpllgmfilhnhkllmkecfmpld?hl=en) - for applying the js to page

## Setup

1. In User Javascript and CSS app, create a new rule and copy contents of [main.js](main.js) into javascript editor
2. For URL pattern setting, set value to `https://www.o2.co.uk/*`
